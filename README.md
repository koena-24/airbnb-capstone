# Airbnb Capstone Project

A full-stack Airbnb-inspired accommodation booking platform developed as my Capstone Project for the Zaio Full Stack Developer Bootcamp.

## Features

### Customer Features

- User Registration
- User Login
- JWT Authentication
- Browse Accommodation Listings
- Search Accommodation
- Filter Listings
- View Property Details
- Dynamic Cost Calculator
- Create Reservations
- View My Reservations
- Leave Reviews
- Responsive Design

### Admin Features

- Secure Login
- Dashboard Statistics
- Create Listings
- Update Listings
- Delete Listings
- View Reservations
- Manage Reviews
- Protected Routes

## Installation

Clone the repository

```bash
git clone https://github.com/koena-24/airbnb-capstone.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Customer Frontend

```bash
cd airbnb-frontend
npm install
npm run dev
```

### Admin Dashboard

```bash
cd admin-frontend
npm install
npm run dev
```
